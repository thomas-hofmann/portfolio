Locker System

  • Es ist nun möglich Dinge mit einem Zahlencode einzuschließen
  • Damit ich den Überblick behalten kann, welche Schränke aktiv genutzt werden, sollte sich bitte jeder der so ein Lockerschrank nutzen will für sein Projekt mit mir bitte vorher kurz besprechen, dann bekommt ihr ein/mehrere Schränke zugeteilt
  • Es gibt auch die Möglichkeit einen der großen Schränke für euer Projekt zu nutzen, da auch bitte mit mir besprechen, dann bekommt ihr ein Schlüssel, welcher dann im Lockerschrank aufbewahrt werden sollte.
  • Achtung: Bitte das untere Video angucken wie die Schränke funktionieren. Man muss den Schrank mit einem Code zu und aufschließen. Danach ist der Schrank wieder resetet und zum schließen muss man wieder den Code eingeben. Ihr solltet euch also mit euren Studis auf einen Code einigen. Sollte man den Code vergessen kann ich den Schrank mit einem Notschlüssel resetten.
